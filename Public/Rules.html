Regras do jogo:


1. Ao iniciar você será redirecionado para tela de seleção.

Jogador 1 escolhe o personagem primeiro e então o jogador 2 pode selecionar.

2. É necessário inserir nomes para cada jogador.

3. Dado início ao jogo, cada jogador responde a uma pergunta. Começando pelo jogador 1.

4. O jogo termina quando todas as perguntas forem respondidas.
